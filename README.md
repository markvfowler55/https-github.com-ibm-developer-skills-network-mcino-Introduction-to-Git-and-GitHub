## Simple Interest Calculator

This project contains a Bash script that calculates simple interest based on user-provided input values for principal amount, rate of interest, and time period.

---

## Description

The script prompts the user to enter:

* Principal amount (p)
* Rate of interest (r)
* Time period in years (t)

It then computes the simple interest using the standard formula.

---

## Formula

Simple Interest = (p × r × t) / 100

---

## Input

```
p: Principal amount  
r: Annual rate of interest  
t: Time period (in years)
```

---

## Output

```
Simple Interest = (p × r × t) / 100
```
