# Code of Conduct

## Our Pledge
We pledge to make participation in this project a harassment-free experience
for everyone.

## Our Standards
We are committed to providing a welcoming and inclusive environment.

## Enforcement Responsibilities
Project maintainers are responsible for enforcing this Code of Conduct.

## Scope
This Code of Conduct applies within all project spaces.

## Enforcement
Instances of abusive, harassing, or unacceptable behavior may be reported
to the project maintainers via GitHub issues or repository discussions.

## Enforcement Guidelines
Maintainers will take appropriate corrective action based on the situation.

## Attribution
This Code of Conduct is adapted from the Contributor Covenant.
